```ts

const token = mintclub.network('base').token('SIGNET')

const [totalSupply, nextMintPrice] = await Promise.all([
  token.getTotalSupply(),
  token.getPriceForNextMint(),
])
```

## One client for the protocol

| Capability | SDK entry point |
|---|---|
| ERC-20 bonding curve assets | `mintclub.network(chain).token(symbolOrAddress)` |
| ERC-1155 bonding curve assets | `mintclub.network(chain).nft(symbolOrAddress)` |
| Airdrops | `mintclub.network(chain).airdrop` |
| Lockups | `mintclub.network(chain).lockup` |
| Staking pools | `mintclub.network(chain).stake` |
| Public and wallet clients | `getPublicClient()`, `withPublicClient()`, `withWalletClient()` |

Read calls work with the SDK's fallback RPC configuration. Write calls require a configured wallet.
