# withProvider

`withProvider` configures a network-scoped wallet client with a custom EIP-1193 provider.

## Usage

:::note[Tip]
The example below uses the `withProvider` method to create a custom [`network`](/docs/sdk/network).
:::

```ts [example.ts]
import { mintclub } from '@mint.club/v2-sdk'

const network = mintclub.network('ethereum')
await network.withProvider(window.ethereum) // [!code focus]

// Subsequent writes on this network use the configured provider.
const walletClient = network.getWalletClient()
```

## Return Value

`Promise<Client>`

## Parameters

### provider

- Type: `any`

Custom wallet provider
