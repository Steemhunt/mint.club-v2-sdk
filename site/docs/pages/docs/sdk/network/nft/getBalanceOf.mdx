# getBalanceOf 

Returns balance of the address for the nft.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const balance = await mintclub.network('ethereum').nft('SYMBOL').getBalanceOf('0x...')
```

## Return Value

`Promise<bigint>`

Returns the balance of the address for the nft.


## Parameters

### walletAddress

- Type: `'0x${string}'`

The wallet address for which the balance is to be fetched.
