# getSymbol 

Returns the symbol of the nft.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const symbol = await mintclub.network('ethereum').nft('SYMBOL').getSymbol()
```

## Return Value

`Promise<string>`

Returns the symbol of the nft.
