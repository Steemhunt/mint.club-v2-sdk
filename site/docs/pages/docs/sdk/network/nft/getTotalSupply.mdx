# getTotalSupply 

Returns the total supply of the nft.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const totalSupply = await mintclub.network('ethereum').nft('SYMBOL').getTotalSupply()
```

## Return Value

`Promise<bigint>`

Returns the total supply of the nft.
