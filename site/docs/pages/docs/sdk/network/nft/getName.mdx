# getName 

Returns the name of the nft.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const name = await mintclub.network('ethereum').nft('SYMBOL').getName()
```

## Return Value

`Promise<string>`

Returns the name of the nft.
