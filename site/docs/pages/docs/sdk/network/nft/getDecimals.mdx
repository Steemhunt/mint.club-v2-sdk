# getDecimals 

Returns the decimals of the nft. For mint.club Bonding Curved nfts, the decimals is always `0`.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const decimals = await mintclub.network('ethereum').nft('SYMBOL').getDecimals()
// 0
```

## Return Value

`Promise<number>`

Returns the decimals of the nft.
