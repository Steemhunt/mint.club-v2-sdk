# getUri 

Returns the URI of the NFT

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const uri = await mintclub.network('ethereum').nft('SYMBOL').getUri()
```

## Return Value

`Promise<string>`

Returns the URI of the NFT
