# exists 

Checks if token exists.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
await mintclub.network('ethereum').token('SYMBOL').exists()
```

## Return Value

`Promise<boolean>`

Returns a promise that resolves to `true` or `false`
