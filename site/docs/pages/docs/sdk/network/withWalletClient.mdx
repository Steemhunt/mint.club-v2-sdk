# withWalletClient

`withWalletClient()` configures SDK writes with an existing viem `WalletClient`. The client must include an account and chain.

```ts
import { mintclub, wei } from '@mint.club/v2-sdk'
import { createWalletClient, custom } from 'viem'
import { base } from 'viem/chains'

const [account] = await window.ethereum.request({
  method: 'eth_requestAccounts',
})

const walletClient = createWalletClient({
  account,
  chain: base,
  transport: custom(window.ethereum),
})

mintclub.withWalletClient(walletClient)

const token = mintclub.network('base').token('SIGNET')
const amount = wei(100, await token.getDecimals())

await token.buy({ amount, slippage: 1 })
```

Wallet clients returned by wagmi's `useWalletClient()` can be passed directly when they contain the selected account and chain.

## Return value

`MintClubSDK`, so root methods such as `network()` can be chained after configuration.

## Parameter

- `walletClient`: a viem [`WalletClient`](https://viem.sh/docs/clients/wallet) with `account` and `chain` configured.

The SDK rejects clients that do not include a chain ID. A write also fails when no account is available.
