# withPublicClient

Override the configuration of default [`PublicClient`](/docs/sdk/network/getPublicClient) with a custom configuration.

## Usage

:::note[Tip]
The example below uses the `withPublicClient` method to create a custom [`network`](/docs/sdk/network).
:::

```ts [example.ts]
import { mintclub } from 'mint.club-v2-sdk';

await mintclub
  .withPublicClient({ // [!code focus]
    chain: 'rick-morty-chain', // [!code focus]
    transport: http('https://rnm-mainnet.g.alchemy.com/v2/...') // [!code focus]
    ...
  }) // [!code focus]
  .token('MINT')
  .create({
    name: 'Mint Club',
    reserveToken: {
      address: '0xC02aaA39b223FE8D0A0e5C4F27eAD9083C756Cc2', // mainnet WETH token address
      decimals: 18,
    },
	curveData: {
		curveType: 'LINEAR',
		stepCount: 10,
		maxSupply: 10_000,
		initialMintingPrice: 0.01, // 0.01 WETH
		finalMintingPrice: 0.1, // 0.1 WETH
		creatorAllocation: 100,
	}
  })
```

## Return Value

[`network`](/docs/sdk/network)

Returns the `network` instance, so you chain functions such as [`token`](/docs/sdk/network/nft) and [`nft`](/docs/sdk/network/nft).

## Parameters

### PublicClient

- Type: [`PublicClient`](https://viem.sh/docs/clients/public)
