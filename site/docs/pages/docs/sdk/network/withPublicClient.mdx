# withPublicClient

`withPublicClient()` registers an existing viem `PublicClient` by its chain ID. Reads on that chain then use the supplied transport.

```ts
import { mintclub } from '@mint.club/v2-sdk'
import { createPublicClient, http } from 'viem'
import { base } from 'viem/chains'

const publicClient = createPublicClient({
  chain: base,
  transport: http(process.env.BASE_RPC_URL),
})

mintclub.withPublicClient(publicClient)

const details = await mintclub
  .network('base')
  .token('SIGNET')
  .getDetail()
```

## Return value

`MintClubSDK`, so root methods such as `network()` can be chained after configuration.

## Parameter

- `publicClient`: a viem [`PublicClient`](https://viem.sh/docs/clients/public) with `chain` configured.

The SDK rejects clients that do not include a chain ID.
