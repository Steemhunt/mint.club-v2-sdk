# getUsdRate 

Returns the estimated UsdRate from OneInch spot aggregator Contract.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const usdRate = await mintclub.network('ethereum').token('SYMBOL').getUsdRate(100)
```

## Return Value

`Promise<[estimatedOutcome: bigint, royalty: bigint]>`


### estimatedOutcome

- Type: `bigint`

Estimated output amount of the token for the given input amount of the token.

### royalty

- Type: `bigint`

Royalty amount given to the creator of the token.


## Parameters

### amount (optional)

- Type: `number`
- Default: `1`

The amount of the token to be converted to USD. If not provided, the default value is `1`.
