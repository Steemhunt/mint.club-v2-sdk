# sell 

Sells tokens from the bonding curve contract.

:::tip[tip]
Before user makes a `sell` call, the bonding curve contract must be `approved` to use the asset's reserve token.

:::success
The SDK will automatically check if the bonding curve contract is approved to use the asset's reserve token. If not approved, it will prompt the user to sign an approval transaction. 
:::

## Usage

```ts twoslash [example.ts]
import { mintclub, wei } from 'mint.club-v2-sdk'
 
await mintclub.network('ethereum').token('SYMBOL').sell({
	amount: wei(1, 18),
})
```

## Return Value

`Promise<TransactionReceipt>`

Returns a promise that resolves to a `TransactionReceipt` object when the transaction is successful.


## Parameters


### amount

- Type: `bigint`

The amount of tokens to sell from the bonding curve contract.

```ts twoslash [example.ts]
import { mintclub, wei } from 'mint.club-v2-sdk'
 
await mintclub.network('ethereum').token('SYMBOL').sell({
	amount: wei(1, 18) // [!code focus]
})
```

### recipient (optional)

- Type: `'0x${string}'`
- Default: `signer address`

The address to send the bought tokens to. If not provided, the tokens will be sent to the user's address.

```ts twoslash [example.ts]
import { mintclub, wei } from 'mint.club-v2-sdk'
 
await mintclub.network('ethereum').token('SYMBOL').sell({
	amount: wei(1, 18),
	recipient: '0x1234567890123456789012345678901234567890' // [!code focus]
})
```

### slippage (optional)

- Type: `number`
- Default: `0`


```ts twoslash [example.ts]
import { mintclub, wei } from 'mint.club-v2-sdk'
 
await mintclub.network('ethereum').token('SYMBOL').sell({
	amount: wei(1, 18),
	slippage: 1 // 1% // [!code focus]
})
```

### allowanceAmount (optional)

- Type: `bigint`
- Default: `MaxUint256`

The amount of tokens to approve the bonding curve contract to use the asset's reserve token. If not provided, the SDK will automatically use `MaxUint256` as the allowance amount.

```ts twoslash [example.ts]
import { mintclub, wei } from 'mint.club-v2-sdk'
 
await mintclub.network('ethereum').token('SYMBOL').sell({
	amount: wei(1, 18),
	allowanceAmount: wei(1, 18) // [!code focus]
})
```



### onAllowanceSignatureRequest (optional)

- Type: `onAllowanceSignatureRequest?: () => void;`
- Default: `undefined`

Callback function for when the user is requested to sign the approval transaction.

```ts
await mintclub
  .network('ethereum')
  .token('MINT')
  .sell({
	 amount: wei(1, 18)
	 onAllowanceSignatureRequest: () => {} // [!code focus]
  })
```


### onAllowanceSigned (optional)

- Type: `onAllowanceSigned?: (txHash: '0x${string}') => void;`
- Default: `undefined`


Callback function for when the user is signs the approval transaction.

```ts
await mintclub
  .network('ethereum')
  .token('MINT')
  .sell({
	 amount: wei(1, 18)
	 onAllowanceSigned: (txHash) => {} // [!code focus]
  })
```

### onAllowanceSuccess (optional)

- Type: `onAllowanceSuccess?: (receipt: TransactionReceipt) => void;`
- Default: `undefined`


Callback function for when the approval transaction is successful.

```ts
await mintclub
  .network('ethereum')
  .token('MINT')
  .sell({
     amount: wei(1, 18)
     onAllowanceSuccess: (receipt) => {} // [!code focus]
  })
```


### onSignatureRequest (optional)

- Type: `onSignatureRequest?: () => void;`
- Default: `undefined`

Callback function for when the user is requested to sign the transaction.

```ts
await mintclub
  .network('ethereum')
  .token('MINT')
  .sell({
	 amount: wei(1, 18)
	 onSignatureRequest: () => {} // [!code focus]
  })
```

### onSigned (optional)

- Type: `onSigned?: (txHash: '0x${string}') => void;`
- Default: `undefined`


Callback function for when the user is signs the transaction.

```ts
await mintclub
  .network('ethereum')
  .token('MINT')
  .sell({
	 amount: wei(1, 18)
	 onSigned: (txHash) => {} // [!code focus]
  })
```

### onSuccess (optional)

- Type: `onSuccess?: (receipt: TransactionReceipt) => void;`
- Default: `undefined`


Callback function for when the transaction is successful.

```ts
await mintclub
  .network('ethereum')
  .token('MINT')
  .sell({
     amount: wei(1, 18)
     onSuccess: (receipt) => {} // [!code focus]
  })
```

### onError (optional)

- Type: `onError?: (error: unknown) => void;`
- Default: `undefined`

Callback function for when the transaction fails. This also includes when the user rejects the transaction.

```ts
await mintclub
  .network('ethereum')
  .token('MINT')
  .sell({
	  amount: wei(1, 18)
	  onError: (error) => {} // [!code focus]
  })
```
