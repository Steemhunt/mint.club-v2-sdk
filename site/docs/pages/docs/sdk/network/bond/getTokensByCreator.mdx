# getTokensByCreator 

Returns tokens created by a user.


## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'
 
const tokens = await mintclub.network('ethereum').bond.getTokensByCreator({
	creator: '0x...',
	start: 0,
	end: 1000
})
```

## Return Value

`Promise<readonly '0x${string}'[]>`

Returns tokens created by a user.

## Parameters

### creator

- Type: `'0x${string}'`  

The address of creator

### start (optional)

- Type: `number`
- Default: `0`

Start index.

### end (optional)

- Type: `number`
- Default: `1000`

End index.

:::warning[Note]
Note that passing 0 and 100 does **NOT** mean that it will fetch the first 100. 

It means that it will check and return the ids from 0 to 100.
:::
