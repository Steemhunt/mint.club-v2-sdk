# claimAirdrop 

Checks is airdrop is public or not.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'

await mintclub.network('base').airdrop.claimAirdrop({
	airdropId: 1
}) 
```



## Return Value

`Promise<boolean>`

- Type: `number`
