# getIsClaimed 

Checks is user claimed the airdrop.

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'

const claimed = await mintclub.network('base').airdrop.getIsClaimed(1, '0x...') 
```



## Return Value

`Promise<boolean>`

## Parameters

### id

- Type: `number`

Airdrop id.

### address

- Type: `'0x${string}'`

User address.
