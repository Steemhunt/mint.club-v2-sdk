# createLockUp 

Create a lockup for a token.

:::tip[tip]
Before user makes a `createLockUp` call, the lockup contract must be `approved` to use the token to lockup.

:::success
The SDK will automatically check if the lockup contract is approved to use the asset. If not approved, it will prompt the user to sign an approval transaction. 
:::

## Usage

```ts twoslash [example.ts]
import { mintclub } from 'mint.club-v2-sdk'

await mintclub.network('base').lockup.createLockUp({
	title: 'Test Lock Up',
	token: '0x...',
	isERC20: true,
	amount: 1000000000000000n,
	unlockTime: Math.floor(Date.now() / 1000) + 60 * 60 * 24 * 7,
	receiver: '0x...',
}) 
```


## Return Value

`Promise<TransactionReceipt>`

Returns a promise that resolves to a `TransactionReceipt` object when the transaction is successful.

The title of the lockup.

