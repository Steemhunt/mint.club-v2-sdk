# uploadMetadata 

Upload a ERC-1155 Metadata to IPFS. 

:::tip[Tip]
You need to have a Filebase API key to use this function. 

Refer to [getting a filebase API Key](/docs/sdk/ipfs#getting-an-api-key).
:::

## Usage

```ts [example.ts]
import { mintclub } from 'mint.club-v2-sdk'

await mintclub.ipfs.uploadMetadata({ 
	filebaseApiKey: 'YOUR API KEY', 
	image: 'ipfs://...',
	name: 'My NFT',
	description: 'My NFT description',
	external_url: 'https://mint.club',
	attributes: [{ trait_type: 'rarity', value: 'legendary' }],
	video: 'ipfs://...'
}) 
```

:::warning[Good to know]
You could also pass a `http` url to the `image` and `video` parameters, if you want to change the content and "reveal" the NFT afterwards.
:::

## Return Value

`Promise<'ipfs://${string}'>`

Returns a promise that resolves to a string containing the IPFS hash of the uploaded file.


## Parameters

- Type: `MetadataUploadParams`

```ts
type MetadataUploadParams = {
  filebaseApiKey: string;
  image: IpfsHashUrl | HttpUrl;
  name: string;
  description?: string;
  external_url?: string;
  attributes?: { trait_type: string; value: string }[];
  video?: IpfsHashUrl | HttpUrl;
};
```
