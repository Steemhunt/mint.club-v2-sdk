# Getting started

The SDK supports browsers and Node.js, ships ESM and CommonJS builds, and uses [viem](https://viem.sh) for blockchain clients.

## Install

:::code-group

```bash [npm]
npm install @mint.club/v2-sdk
```

```bash [pnpm]
pnpm add @mint.club/v2-sdk
```

```bash [yarn]
yarn add @mint.club/v2-sdk
```

```bash [bun]
bun add @mint.club/v2-sdk
```

:::

## Read protocol data

Read methods use the SDK's fallback RPCs, so they do not require a wallet.

```ts
import { mintclub } from '@mint.club/v2-sdk'

const token = mintclub.network('base').token('CHICKEN')

const [totalSupply, nextMintPrice] = await Promise.all([
  token.getTotalSupply(),
  token.getPriceForNextMint(),
])
```

`network()` accepts either a supported lowercase network key such as `base` or its numeric chain ID, `8453`.

## Configure writes

Use `withWalletClient()` with a viem-compatible wallet client in applications. A wallet client must include both an account and a chain.

```ts
import { mintclub, wei } from '@mint.club/v2-sdk'
import { createWalletClient, custom } from 'viem'
import { base } from 'viem/chains'

const [account] = await window.ethereum.request({ method: 'eth_requestAccounts' })
const walletClient = createWalletClient({
  account,
  chain: base,
  transport: custom(window.ethereum),
})

mintclub.withWalletClient(walletClient)

const chicken = mintclub.network('base').token('CHICKEN')
const amount = wei(100, await chicken.getDecimals())

await chicken.buy({
  amount,
  slippage: 1, // 1%
})
```

For a trusted server process, select the network before calling `withPrivateKey()`:

```ts
import { mintclub, wei } from '@mint.club/v2-sdk'

const serverToken = mintclub
  .network('base')
  .withPrivateKey(process.env.PRIVATE_KEY as `0x${string}`)
  .token('CHICKEN')

const serverAmount = wei(100, await serverToken.getDecimals())
await serverToken.buy({ amount: serverAmount, slippage: 1 })
```

Never place a private key in frontend code, logs, command arguments, or source control.
