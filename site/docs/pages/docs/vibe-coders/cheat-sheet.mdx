# SDK quick reference

## Core pattern

```ts
import { mintclub } from '@mint.club/v2-sdk'

const asset = mintclub
  .network('base')
  .token('SIGNET')

const details = await asset.getDetail()
```

Use `token()` for ERC-20 assets and `nft()` for ERC-1155 assets. Each accepts a Mint Club symbol or contract address.

## Configure a wallet

For applications, provide a viem wallet client with an account and chain:

```ts
mintclub.withWalletClient(walletClient)

await mintclub.network('base').token('SIGNET').buy({
  amount: 100n,
  slippage: 1,
})
```

For trusted server processes only:

```ts
await mintclub
  .network('base')
  .withPrivateKey(process.env.PRIVATE_KEY as `0x${string}`)
  .token('SIGNET')
  .buy({ amount: 100n, slippage: 1 })
```

## Amounts and percentages

- Amounts are `bigint` values in the asset's smallest unit.
- Query ERC-20 decimals before converting a human-readable amount.
- Mint Club ERC-1155 assets use whole units such as `1n`.
- `slippage: 1` means 1%; it is not a basis-point value.
- `buyRoyalty: 3` and `sellRoyalty: 3` mean 3%.

```ts
import { wei } from '@mint.club/v2-sdk'

const token = mintclub.network('base').token('SIGNET')
const decimals = await token.getDecimals()
const amount = wei(100, decimals)
```

## Read and estimate

```ts
await token.exists()
await token.getDetail()
await token.getSteps()
await token.getTotalSupply()
await token.getReserveTokenAddress()
await token.getPriceForNextMint()
await token.getBuyEstimation(amount)
await token.getSellEstimation(amount)
await token.getUsdRate({ amount: 1 })
```

## Buy and sell

```ts
await token.buy({
  amount,
  slippage: 1,
  recipient: '0x...',
  onSigned: (hash) => console.log(hash),
  onSuccess: (receipt) => console.log(receipt.transactionHash),
  onError: (error) => console.error(error),
})

await token.sell({ amount, slippage: 1 })
```

The SDK checks the relevant allowance and may request an approval signature before the trade signature.

## Create an ERC-20 asset

```ts
await mintclub.network('base').token('MYTOKEN').create({
  name: 'My Token',
  reserveToken: {
    address: '0x4200000000000000000000000000000000000006',
    decimals: 18,
  },
  curveData: {
    curveType: 'EXPONENTIAL',
    stepCount: 100,
    maxSupply: 1_000_000,
    initialMintingPrice: 0.0001,
    finalMintingPrice: 0.1,
    creatorAllocation: 100,
  },
  buyRoyalty: 3,
  sellRoyalty: 3,
})
```

Pass `stepData` instead of `curveData` to supply explicit cumulative ranges and prices.

## Create an ERC-1155 asset

```ts
await mintclub.network('base').nft('MYNFT').create({
  name: 'My Collection',
  metadataUrl: 'ipfs://...',
  reserveToken: { address: '0x...', decimals: 18 },
  stepData: [
    { rangeTo: 100, price: 0 },
    { rangeTo: 1_000, price: 0.01 },
  ],
})
```

## Protocol helpers

```ts
const network = mintclub.network('base')

network.bond
network.airdrop
network.lockup
network.stake
network.getPublicClient()
network.getWalletClient()
```

Optional protocol contracts differ by network. Resolve and validate the address before constructing a write.
