## Core pattern

```ts
import { mintclub } from '@mint.club/v2-sdk'

const asset = mintclub
  .network('base')
  .token('SIGNET')

const details = await asset.getDetail()
```

Use `token()` for ERC-20 assets and `nft()` for ERC-1155 assets. Reads do not require a wallet. Configure a wallet client before writes, and always query decimals and current estimates instead of guessing units.

For a compact machine-readable reference, use the repository's [`llms-full.txt`](https://github.com/Steemhunt/mint.club-v2-sdk/blob/main/llms-full.txt).
