# 🤖 Complete LLM Guide for Mint Club SDK

> **For AI Assistants:** This is your comprehensive reference for the Mint Club V2 SDK. Study this document thoroughly before implementing any code.

## ⚡ The Core Pattern (Memorize This!)

The Mint Club SDK follows a simple, consistent pattern:

```ts
import { mintclub } from 'mint.club-v2-sdk'

mintclub           // 1. Start with mintclub
  .network('base') // 2. Select network
  .token('SYMBOL') // 3. Choose token or nft
  .buy({...})      // 4. Call function
```

**ALWAYS follow this pattern:**

1. **mintclub** - The main SDK entry point
2. **network()** - Select blockchain network
3. **token()** or **nft()** - Choose asset to interact with
4. **function()** - Call the desired operation

## 🔑 Wallet Integration (Critical for AI)

### ❌ WRONG: Don't use sdk.wallet

```ts
// DON'T DO THIS - sdk.wallet is not recommended
await sdk.wallet.connect()
await sdk.wallet.account()
```

### ✅ CORRECT: Use withWalletClient

**Node.js Backend:**

```ts
import { privateKeyToAccount } from 'viem/accounts'
import { createWalletClient, http } from 'viem'
import { base } from 'viem/chains'

const account = privateKeyToAccount(process.env.PRIVATE_KEY!)
const walletClient = createWalletClient({
  account,
  chain: base,
  transport: http(),
})

const sdk = mintclub.withWalletClient(walletClient)
```

**React/Next.js with wagmi:**

```ts
import { useWalletClient } from 'wagmi'
import { base } from 'viem/chains'

function MyComponent() {
  const { data: walletClient } = useWalletClient({ chainId: 8453 })

  const sdk = walletClient ? mintclub.withWalletClient({ ...walletClient, chain: base }) : mintclub // fallback to read-only

  // Use sdk here
}
```

## 💰 Token Operations (Most Common)

### Creating Tokens

```ts
// ERC20 Token
await sdk
  .network('base')
  .token('MYTOKEN')
  .create({
    name: 'My Token',
    reserveToken: { address: '0x4200000000000000000000000000000000000006', decimals: 18 },
    curveData: {
      curveType: 'LINEAR',
      stepCount: 20,
      maxSupply: 1_000_000,
      initialMintingPrice: 0.001,
      finalMintingPrice: 1,
    },
    buyRoyalty: 500, // 5%
    sellRoyalty: 300, // 3%
  })

// ERC1155 NFT
await sdk
  .network('base')
  .nft('MYART')
  .create({
    name: 'My NFT Collection',
    metadataUrl: 'ipfs://...',
    reserveToken: { address: '0x4200000000000000000000000000000000000006', decimals: 18 },
    curveData: {
      /* same as above */
    },
  })
```

### Trading Tokens

```ts
// Buy tokens
await sdk
  .network('base')
  .token('CHICKEN')
  .buy({
    amount: wei(100, 18), // 100 tokens
    slippage: 100, // 1% slippage
    onSuccess: (receipt) => console.log('Bought!', receipt),
  })

// Sell tokens
await sdk
  .network('base')
  .token('CHICKEN')
  .sell({
    amount: wei(50, 18), // 50 tokens
    slippage: 100,
    onSuccess: (receipt) => console.log('Sold!', receipt),
  })

// Buy with ETH (Zap)
await sdk
  .network('base')
  .token('CHICKEN')
  .buyWithZap({
    amount: wei(10, 18),
    slippage: 200, // Higher slippage for zap
    onSuccess: (receipt) => console.log('Zap purchase!', receipt),
  })
```

### Reading Token Data


## 🥩 Staking Operations

### Create Staking Pool

```ts

  stakingToken: '0x...', // Token to stake
  isStakingTokenERC20: true,
  rewardToken: '0x...', // Token for rewards
```

## ⚠️ Common Mistakes (For AI Assistants)

### 1. Wrong Pattern Order

```ts
// ❌ WRONG
const token = mintclub.token('SYMBOL')
await token.network('base').buy()

// ✅ CORRECT
const token = mintclub.network('base').token('SYMBOL')
await token.buy()
```

### 2. Missing Wallet Setup

```ts
// ❌ WRONG - No wallet configured
await mintclub.network('base').token('CHICKEN').buy()

// ✅ CORRECT - Wallet configured first
const sdk = mintclub.withWalletClient(walletClient)
await sdk.network('base').token('CHICKEN').buy()
```

### 3. Wrong Decimals Handling

```ts
// ❌ WRONG - Not handling decimals
await token.buy({ amount: 100 })

// ✅ CORRECT - Use wei() helper
await token.buy({ amount: wei(100, 18) })
```

### 4. Time Units

```ts
// ❌ WRONG - Using milliseconds
rewardStartsAt: Date.now() + 3600000

// ✅ CORRECT - Using seconds
rewardStartsAt: Math.floor(Date.now() / 1000) + 3600
```

## 🛠️ Error Handling

```ts
import {
  ChainNotSupportedError,
  WalletNotConnectedError,
  TokenAlreadyExistsError,
  InvalidImageProvidedError
} from 'mint.club-v2-sdk'

try {
  await sdk.network('base').token('CHICKEN').buy({...})
} catch (error) {
  if (error instanceof ChainNotSupportedError) {
    console.log('Network not supported')
  } else if (error instanceof WalletNotConnectedError) {
    console.log('Connect wallet first')
  } else {
    console.error('Unexpected error:', error)
  }
}
```

## 📋 Quick Reference

### Core Pattern Steps:

1. **mintclub** - Start here
2. **network()** - Select network
3. **token()/nft()** - Choose asset
4. **function()** - Call operation

### Essential Helpers:

- `wei(amount, decimals)` - Convert to proper units
- `Math.floor(Date.now() / 1000)` - Convert to seconds
- `withWalletClient()` - Setup wallet (required for writes)

### Common Operations:

- `buy()`, `sell()` - Trade tokens
- `create()` - Create new tokens
- `getTotalSupply()`, `getPriceForNextMint()` - Read data
- `createPool()`, `stake()`, `claim()` - Staking operations
