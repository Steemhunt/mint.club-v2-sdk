# Common patterns

## Parallel reads

```ts
import { mintclub } from '@mint.club/v2-sdk'

const token = mintclub.network('base').token('SIGNET')

const [details, steps, totalSupply, usd] = await Promise.all([
  token.getDetail(),
  token.getSteps(),
  token.getTotalSupply(),
  token.getUsdRate({ amount: 1 }),
])
```

## Wallet-backed write

```ts
import { mintclub, wei } from '@mint.club/v2-sdk'

mintclub.withWalletClient(walletClient)

const token = mintclub.network('base').token('SIGNET')
const amount = wei(100, await token.getDecimals())

await token.buy({
  amount,
  slippage: 1,
  onAllowanceSignatureRequest: () => showStatus('Approve reserve token'),
  onSignatureRequest: () => showStatus('Confirm purchase'),
  onSuccess: (receipt) => showStatus(receipt.transactionHash),
  onError: (error) => showError(error),
})
```

The SDK can submit an approval before the trade. Present both signatures to the user as separate actions.

## Trusted server write

```ts
const token = mintclub
  .network('base')
  .withPrivateKey(process.env.PRIVATE_KEY as `0x${string}`)
  .token('SIGNET')

const amount = wei(100, await token.getDecimals())
await token.sell({ amount, slippage: 1 })
```

Never use a private key in frontend code or accept one through an LLM prompt.

## Explicit curve steps

```ts
await mintclub.network('base').token('MYTOKEN').create({
  name: 'My Token',
  reserveToken: { address: '0x...', decimals: 18 },
  stepData: [
    { rangeTo: 100, price: 0 },
    { rangeTo: 1_000, price: 0.01 },
    { rangeTo: 10_000, price: 0.1 },
  ],
  buyRoyalty: 3,
  sellRoyalty: 3,
})
```

`rangeTo` values are cumulative. Pass `stepData` or `curveData`, never both. Royalty values use percentage points.

## Custom public client

```ts
import { createPublicClient, http } from 'viem'
import { base } from 'viem/chains'

const publicClient = createPublicClient({
  chain: base,
  transport: http(process.env.BASE_RPC_URL),
})

mintclub.withPublicClient(publicClient)
const details = await mintclub.network('base').token('SIGNET').getDetail()
```

## Low-level contract read

```ts
import { bondContract } from '@mint.club/v2-sdk'

const creationFee = await bondContract.network('base').read({
  functionName: 'creationFee',
})
```

Prefer high-level asset and protocol helpers unless the required method is not exposed there.
