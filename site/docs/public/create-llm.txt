# Mint Club V2 SDK — asset creation reference

Package: `@mint.club/v2-sdk`

Create ERC-20 assets with `token(symbol).create()` and ERC-1155 assets with `nft(symbol).create()`. Creation is a write and requires a configured wallet.

```ts
mintclub.withWalletClient(walletClient)
```

## Shared parameters

- `name: string`
- `reserveToken: { address: 0x-prefixed address; decimals: number }`
- exactly one of `curveData` or `stepData`
- optional `buyRoyalty` and `sellRoyalty` in percentage points
- optional write lifecycle callbacks

`buyRoyalty: 3` means 3%. The SDK default `0.03` means 0.03%, not 3%.

## Generated curve

```ts
import { mintclub } from '@mint.club/v2-sdk'

await mintclub.network('base').token('MYTOKEN').create({
  name: 'My Token',
  reserveToken: {
    address: '0x4200000000000000000000000000000000000006',
    decimals: 18,
  },
  curveData: {
    curveType: 'EXPONENTIAL',
    stepCount: 100,
    maxSupply: 1_000_000,
    initialMintingPrice: 0.0001,
    finalMintingPrice: 0.1,
    creatorAllocation: 100,
  },
  buyRoyalty: 3,
  sellRoyalty: 3,
  onSuccess: (receipt) => console.log(receipt.transactionHash),
  onError: (error) => console.error(error),
})
```

`creatorAllocation` creates an initial zero-price range allocated to the creator.

## Explicit steps

`rangeTo` values are cumulative and `price` is denominated in the reserve token.

```ts
await mintclub.network('base').token('MYTOKEN').create({
  name: 'My Token',
  reserveToken: { address: '0x...', decimals: 18 },
  stepData: [
    { rangeTo: 100, price: 0 },
    { rangeTo: 1_000, price: 0.01 },
    { rangeTo: 10_000, price: 0.1 },
  ],
})
```

Do not pass both `curveData` and `stepData`.

## ERC-1155 asset

ERC-1155 creation also requires `metadataUrl`:

```ts
await mintclub.network('base').nft('MYNFT').create({
  name: 'My Collection',
  metadataUrl: 'ipfs://...',
  reserveToken: { address: '0x...', decimals: 18 },
  stepData: [{ rangeTo: 1_000, price: 0.01 }],
})
```

The SDK validates `ipfs://` hashes. An HTTPS metadata URL is also accepted.

## Safety

- Confirm the network, symbol, reserve address, reserve decimals, curve, royalties, and creator allocation before signing.
- Check `await asset.exists()` to avoid reusing an existing symbol.
- Do not guess reserve decimals.
- Treat creation fees and gas estimates as live values.
- Some write helpers report failures through `onError` and return `undefined`.
