# Mint Club V2 SDK — staking reference

Package: `@mint.club/v2-sdk`

Use `mintclub.network(chain).stake`. Staking is an optional contract feature and is not deployed on every supported network.

## Validate deployment and limits

```ts
import { getMintClubContractAddress, mintclub, supportedChainsMap, wei } from '@mint.club/v2-sdk'

const network = mintclub.network('base')
const stake = network.stake
const stakeAddress = getMintClubContractAddress('STAKE', supportedChainsMap.base)

if (stakeAddress === '0x') throw new Error('Staking is unavailable on this network')

const [creationFee, claimFee, minDuration, maxDuration] = await Promise.all([
  stake.getCreationFee(),
  stake.getClaimFee(),
  stake.getMinRewardDuration(),
  stake.getMaxRewardDuration(),
])
```

Fees and duration limits are live contract values; do not hard-code them.

## Create a pool

Approve the Stake contract to spend the reward token before creating a pool.

```ts
const rewardToken = network.token('REWARD')
const rewardDecimals = await rewardToken.getDecimals()
const rewardAmount = wei(1_000, rewardDecimals)

await rewardToken.approve({ spender: stakeAddress, amount: rewardAmount })

await stake.createPool({
  stakingToken: '0x...',
  isStakingTokenERC20: true,
  rewardToken: rewardToken.getTokenAddress(),
  rewardAmount,
  rewardStartsAt: Math.floor(Date.now() / 1000) + 3600,
  rewardDuration: 7 * 24 * 60 * 60,
  value: creationFee,
})
```

For an ERC-1155 staking asset, use whole-unit amounts and operator approval. Reward tokens are ERC-20 assets.

## Stake and unstake

Approve the Stake contract to spend the staking asset before calling `stake()`.

```ts
const stakingToken = network.token('STAKE-TOKEN')
const stakingDecimals = await stakingToken.getDecimals()
const amount = wei(100, stakingDecimals)

await stakingToken.approve({ spender: stakeAddress, amount })
await stake.stake({ poolId: 1, amount })
await stake.unstake({ poolId: 1, amount })
```

## Claim and read

```ts
const account = '0x1111111111111111111111111111111111111111'

await stake.getPool({ poolId: 1 })
await stake.getUserPoolStake({ user: account, poolId: 1 })
await stake.getClaimableReward({ staker: account, poolId: 1 })

await stake.claim({ poolId: 1, value: claimFee })
```

Other methods include `getPools()`, `getPoolsByCreator()`, `getClaimableRewardBulk()`, `cancelPool()`, and `emergencyUnstake()`.

## Safety

- Confirm the pool ID, token types, decimals, reward start, duration, and live fees.
- Validate the Stake address and reject `0x`.
- Approvals are separate transactions; the staking helper does not submit them automatically.
- Administrative methods such as fee updates and ownership transfer require the contract owner.
